# Technicians Module

## Description

Technician management and assignment module.

## Features

- Technician listing
- Availability tracking
- Performance metrics
- Order assignment

## Endpoints

| Method | Path                      | Description            |
| ------ | ------------------------- | ---------------------- |
| GET    | `/technicians`            | List technicians       |
| GET    | `/technicians/:id`        | Get technician details |
| GET    | `/technicians/:id/orders` | Technician's orders    |
| GET    | `/technicians/available`  | Available technicians  |
