## [1.0.0] - 2026-01-17

### Added

- Sistema de autenticación 2FA con TOTP
- Módulo de gestión de órdenes de trabajo
- Dashboard administrativo con KPIs
- API REST con documentación Swagger
- Sistema de notificaciones y alertas
- Gestión de técnicos y asignaciones
- Generación de reportes PDF
- Sistema de formularios dinámicos

### Security

- Implementación de JWT con refresh tokens
- Rate limiting en endpoints públicos
- Validación de schemas con class-validator
- Autenticación two-factor (2FA)
