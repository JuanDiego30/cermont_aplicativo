# Usuarios Module

## Description
User management and profile operations.

## Features
- User CRUD operations
- Profile management
- Role-based filtering
- Avatar management

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | `/usuarios` | List users |
| GET | `/usuarios/:id` | Get user details |
| PATCH | `/usuarios/:id` | Update user |
| DELETE | `/usuarios/:id` | Deactivate user |
| GET | `/usuarios/profile` | Current user profile |
