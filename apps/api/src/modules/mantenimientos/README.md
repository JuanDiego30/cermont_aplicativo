# Mantenimientos Module

## Description
Equipment maintenance scheduling and tracking.

## Features
- Scheduled maintenance
- Maintenance history
- Technician assignment
- Equipment health tracking

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | `/mantenimientos` | List maintenance |
| POST | `/mantenimientos` | Create maintenance |
| GET | `/mantenimientos/:id` | Get details |
| PATCH | `/mantenimientos/:id` | Update status |
