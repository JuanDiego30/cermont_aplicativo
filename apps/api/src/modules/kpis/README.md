# KPIs Module

## Description
Key Performance Indicators calculation and tracking.

## Features
- Performance metrics
- Trend analysis
- Goal tracking
- Cached calculations

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | `/kpis` | Get all KPIs |
| GET | `/kpis/tecnicos` | Technician KPIs |
| GET | `/kpis/ordenes` | Order KPIs |
