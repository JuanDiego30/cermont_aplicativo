# Kits Module

## Description
Typical kit management for planning standardization.

## Features
- Kit templates
- Item management
- Planning association
- Category organization

## Endpoints
| Method | Path | Description |
|--------|------|-------------|
| GET | `/kits` | List all kits |
| POST | `/kits` | Create kit |
| GET | `/kits/:id` | Get kit details |
| PATCH | `/kits/:id` | Update kit |
